What you will find on this webpage:

On this website, you will find a detailed explanation of the importance of accessibility in website usage, as well as ways I have made this page accessible, and other references for building an accessible webpage. This site will explain the importance of accessibility in web design, as well as provide information about color contrast adhering to the Web Content Accessibility Guidelines (WCAG)'s standards, and how to incorporate them in your work. What is Web Accessibility?

Web accessibility is the idea that a webpage can be used with ease for any person, regardless of their personal abilities or experience level. The goal is to make an interface that is intuitive, easy to navigate, and has features that allow those with disabilities to operate the page effectively.

What features of this webpage demonstrate web accessibility?

Within this page, there is a detailed description of the importance of accessibility in web design, as well as why it matters, and some sources to allow you to learn more about these topics. Additionally, there is an example of a color contrast guide and a video tutorial with closed captioning to demonstrate the effectiveness of tutorial videos with accessible features.
